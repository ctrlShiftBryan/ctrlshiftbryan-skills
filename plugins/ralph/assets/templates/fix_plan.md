# fix_plan.md

Ordered checklist of work items. Ralph picks the first unchecked item every
loop. When all boxes are checked, Ralph emits `<promise>COMPLETE</promise>`
and the wrapper opens a draft PR.

If an item references a GitHub issue, Ralph runs
`gh issue view <N>` to read the full spec.
